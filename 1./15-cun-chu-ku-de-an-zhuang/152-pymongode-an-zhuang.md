* 官方文档：[https://api.mongodb.com/python/current/](https://api.mongodb.com/python/current/)
* PyPi：[https://pypi.python.org/pypi/pymongo](https://pypi.python.org/pypi/pymongo)

### 2.安装

```
pip install pymongo
```

### 3.验证安装

```
python3

>>> import pymongo
>>> pymongo.version
'3.7.0'
```

如果输出类似上面的内容，就代表pymongo安装成功
