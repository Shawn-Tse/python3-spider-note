Gerapy 是一个 Scrapy 分布式管理模块，本节来介绍一下 Gerapy 的安装方式。

### 1. 相关链接 {#1-相关链接}

* GitHub：
  [https://github.com/Gerapy](https://github.com/Gerapy)

### 2. Pip安装 {#2-pip安装}

推荐使用 Pip 安装，命令如下：

```
pip3 install gerapy

```

### 3. 测试安装 {#3-测试安装}

安装完成之后，可以在 Python 命令行下测试。

```
python3
>>> import gerapy
```

如果没有错误报出，则证明库已经安装好了。
